# Mindee API PHP


## cURL requests

### cURL setup

Run the following command to install the basic tools for php:
```bash
sudo apt install software-properties-common
sudo add-apt-repository ppa:ondrej/php
sudo apt update && sudo apt upgrade
sudo apt install php7.3 php7.3-cli php7.3-curl
```

To start php without having to install the apache module, you can install the `php-cgi` package:
```bash
sudo apt install php-cgi
```

Install composer by following [this tutorial](https://www.digitalocean.com/community/tutorials/how-to-install-and-use-composer-on-ubuntu-20-04).
